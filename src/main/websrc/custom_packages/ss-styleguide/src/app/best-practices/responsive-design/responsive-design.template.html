Responsive Design

Responsive design refers to the practive of designing sites and applications on the web that respond, through the use of media queries, to the size of the screen that the user is using to access the site, or application.

Mobile First Design

When creating sites that are designed to respond to the the device on which they are being displayed, it is common best practice to design 'mobile first', or using a small screen to large screen methodology. This allows the designer to initally concentrate on the hierachical structure of the information, so that the most important, or useful, information is displayed in a prominant area.

Why design for mobile users?

Mobile device's have been outselling dektop and laptop computer for years. More users access web services now than ever before, and this is a trend that is continuing to grow. As more users access these services on their devices, they expect that the services will be optimise to run on their device.

There are a number of solutions to achieve this, but the most widely used and supported solution is the responsive approach to web design.

Fluid and Breakpoints

Responsive design works by using percentage values for setting widths of elements so that they stretch to fill areas of the screen and responds to the overall width of the viewport. Media queries are also used to call various CSS styles that are used specifically above the designated screen widths set in the media queries.
